# Development Workflow Convention

Apply these rules when you are working on this project.

- MUST run `npm run format` after changes.
- MUST run `npm run lint` to verify the code quality after changes.
- MUST run `npm run test:e2e` to verify the end-to-end tests when changes are made to the UI.
